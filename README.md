# Global Weather Trend Forecasting

Exploratory analysis and forecasting of global weather patterns using the
[Global Weather Repository](https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository) dataset.

## Dataset

Source: Kaggle — `nelgiriyewithana/global-weather-repository`

Place `GlobalWeatherRepository.csv` in `data/raw/` before running.

## Setup

```bash
pip install -r requirements.txt
```
